Place your royalty-free background music tracks here.

Recommended structure (mood folders):
- bgm_library/adventure/
- bgm_library/mystery/
- bgm_library/uplifting/
- bgm_library/calm/

Supported file types:
- .mp3 .wav .m4a .aac .flac .ogg .opus

Selection behavior:
1) Prefer files inside bgm_library/<mood>/
2) Else prefer files with mood keyword in filename
3) Else pick from any track in bgm_library/

Optional environment override:
- LASE_BGM_LIBRARY_DIR=/absolute/path/to/your/music/folder

License notes:
- Keep license/attribution files beside each track if required.
- Ensure the track license allows your target use (e.g., YouTube monetization).
